📖 About FraudShield AI


FraudShield AI is an enterprise-inspired fraud detection platform built using Machine Learning and FastAPI.


It analyzes financial transactions in real time using an XGBoost model trained on over 6.3 million PaySim transactions.


The system predicts fraud probability, assigns a risk level, highlights important risk factors and generates a plain-language explanation automatically.


🔗 View source on GitHub

⚙️ How It Works


Every transaction moves through the same five-stage pipeline, end to end in under 100ms:

1

Transaction submitted via API

2

11 features engineered (log-scaled balances, drain & anomaly flags, ratios)

3

XGBoost scores fraud probability

4

Risk level assigned (Low → Critical)

5

A rule-based engine writes a plain-English summary

🚧 Limitations & Roadmap


The current model is trained on the synthetic PaySim dataset, which mimics real transaction patterns but isn't real financial data — accuracy numbers here reflect performance on synthetic fraud, not a production population.


Planned next steps:

  • Validate against a real-world fraud dataset (e.g. IEEE-CIS)
  • Re-check the decision threshold if the production class balance shifts — a 0.5 cutoff currently outperforms the F1-optimal precision-recall threshold on this dataset, but that can change with different data
  • Add authentication and rate limiting before any real deployment
